lme4ord

Generalized linear mixed models with structured covariance structures, with ecologists in mind.
The lme4 in lme4ord is a reference to this mixed models package.
The ord in lme4ord is a reference to model-based ordination, which is an ecological data analysis tool.
Although lme4ord was originally developed for my own ecological applications it can be used much more broadly.
